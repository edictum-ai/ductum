What works better now

  • Crash recovery is durable enough to survive an API kill and classify an expired lease as resumable.
  • Stale owners are fenced out of recovery-critical writes.
  • Terminal runs are not silently overwritten by lineage cleanup.
  • Checkpoint resume keeps the source run's runtime agent when a task was manually started on a different agent than its assignment.
  • Deterministic poison quarantine is now dogfoodable through a packaged mock-agent sample spec.
  • Cost surfaces no longer pretend unknown usage is free.
  • The dashboard has one canonical dark token layer instead of split status color maps.
  • Local dashboard launch now uses the existing one-shot welcome handoff path instead of making token paste the happy path.
  • The fallback UI now says reconnect/manual access rather than centering the operator token.
  • Home now opens on the work that needs the operator, including quarantined and frozen runs with shared status tones.
  • RunDetail no longer splits approve/retry/cancel across separate panels; shipped mutating actions now share one reasoned intervention surface and a parity manifest.
  • RunDetail now leads with a single newest-first timeline instead of making the operator reconcile six raw tabs by hand.

Bugs found while dogfooding

  • Remote dashboard auth still needs a real non-token login or pairing story; manual access is improved but remains a fallback.
  • The operator token is still a bad human-facing UX. It should stay as an automation credential, but dashboard users should get a session or pairing flow.
  • Codex HTTP MCP auth returned Operator token required against the protected local API.
  • Fixed: checkpoint resume used an inaccessible glm agent after the crashed run used opus.
  • Fixed: ductum start stripped DUCTUM_MOCK_AGENT_CALLS=1, so mock proof servers silently loaded real harnesses.
  • Hard-killing the API can leave child harness processes for cleanup/recovery to handle.
  • Fixed while implementing controls: the added RunDetail tests exceeded the 300 LOC gate, so interaction coverage was split into a focused controls test.
  • Fixed while implementing timeline: activity rows could duplicate the same redacted raw text as both title and detail; the timeline now omits redundant detail.
